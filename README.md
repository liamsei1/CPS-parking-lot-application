# Running:
  1) change the password in hibernate file to your database password. the file located in:      CPS\server\src\main\java\resources.
  2) run Mysql Workbench
  4) run in cmd the server jar file located in server.zip
  5) run in cmd the client jar file located in client.zip
  6) enter in the ip field: localhost and in the port field: 3000
